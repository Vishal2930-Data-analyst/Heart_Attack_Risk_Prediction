## ❤️ Heart Attack Prediction System
## 📌 Project Overview

This project is a Machine Learning-based web application that predicts the likelihood of a heart attack using medical parameters such as age, blood pressure, heart rate, blood sugar, CK-MB, and troponin. The model is trained using a Random Forest Classifier and deployed using Streamlit.

## 🎯 Objective

To build a predictive model that helps in early detection of heart attack risk and assists healthcare professionals in making informed decisions.

## 📊 Dataset Features

- Age

- Gender

- Heart Rate

- Systolic Blood Pressure

- Diastolic Blood Pressure

- Blood Sugar

- CK-MB

- Troponin

- Result (Target Variable)

## ⚙️ Technologies Used

- Python

- NumPy

- Pandas

- Scikit-learn

- Matplotlib & Seaborn

- Streamlit

## 🤖 Machine Learning Model

- Algorithm Used: Random Forest Classifier

- Problem Type: Binary Classification

## Output:

- 0 → No Heart Attack

- 1 → High Risk of Heart Attack

## 📈 Project Workflow

- Data Collection

- Data Preprocessing

- Exploratory Data Analysis (EDA)

- Feature Selection

- Model Training

- Model Evaluation

- Model Deployment using Streamlit

## 🖥️ Web App Features

- User-friendly interface

- Real-time prediction

- Medical parameter input

- Instant risk result display

## 📌 Sample Input

- Age: 25
